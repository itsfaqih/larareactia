## Installation
1. Clone this repo.
```bash
git clone https://github.com/itsfaqih/laravel-react-inertiajs-tailwind.git
cd laravel-react-inertiajs-tailwind
```
2. Install the dependencies.
```bash
composer install
npm install
npm run dev
```
3. Copy and setup the .env
```bash
cp .env.example .env
```
4. Migrate the table
```bash
php artisan migrate
```
5. Done! You're ready to go :)
```bash
php artisan serve
```
